Component Blog
==============

The following component is written in [loopback][1] (version 3).

## Purpose

The major purpose of writing this component is to allow an application to:

1. Create a blog post that can be:
  - Text
  - Multiple Images
  - Video
  - Content

2. Users then can:
  - Like
  - Comment
  - Share
  - Mention other users

[1]: https://loopback.io/doc/en/lb3/
